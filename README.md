# AI Sentiment Analyzer
A Python-based tool that uses the TextBlob library to perform Natural Language Processing (NLP). It classifies user input into Positive, Negative, or Neutral categories.

## Features:
- Real-time text analysis.
- Polarity scoring.
- Simple Command Line Interface (CLI).

## How to Run:
1. Install requirements: `pip install textblob`
2. Run the script: `python main.py`
