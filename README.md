# 🤖 AI Sentiment Analyzer

A Python-based Natural Language Processing (NLP) tool that determines the emotional tone of any given text.

## 🚀 Live Demo
You can run this project directly in your browser without installing anything:
[👉 Click here to Run on Google Colab](https://colab.research.google.com/drive/1jlVkXNtE0gmJV_nGoIY-CYbfPu71jCDo?usp=sharing)

## 🛠 Features
- Uses **TextBlob** for semantic analysis.
- Instant classification into Positive, Neutral, or Negative.
- Simple, clean Python logic.

## 💻 Local Installation
If you want to run it on your machine:
1. Clone the repo:https://github.com/alishba1299/AI-Sentiment-Analyzer
2. Install requirements: `pip install textblob`
3. Run: `python main.py`
